John Conway's Game of Life

WIP personal project by Ebrahim Bakhtar

The game of life is a cellular automaton created by the Mathematician John Conway in 1970. From an intial seed, the grid evolves with each generation, each cell interacting with its eight neighbours to determine its next state. There are four rules:

  • Any live cell with fewer than two live neighbors dies, as if by underpopulation.
  • Any live cell with two or three live neighbors lives on to the next generation.
  • Any live cell with more than three live neighbors dies, as if by overpopulation.
  • Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction.
Clicking Reset will generate a random seed which allocates each cell alive or dead.
